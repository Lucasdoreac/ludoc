# last30days
Use esta skill sempre que o usuário ou o fluxo do chat solicitar pesquisas em tempo real sobre tendências, novidades de mercado, ferramentas concorrentes (ex: Cursor vs Windsurf) ou sentimentos da comunidade de desenvolvedores.
Ela agregará dados sociais e gerará um sumário denso com base nos últimos 30 dias de internet.
